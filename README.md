Installation Instructions:
<br />Windows

    1. Install Ruby from this website https://www.ruby-lang.org/en/documentation/installation/
        https://rubyinstaller.org/downloads/archives/

    2. In a command line run "gem install bundler"

    3. Navigate to the root directory of the website and run "bundle install" in a command line

    4. Start the live server by running 
    bundle exec jekyll serve --open-url --livereload --trace

    Because you appended the livereload tag, every time you save a new file it treats it like a pseudo-commit on the local serve, automatically updating it
